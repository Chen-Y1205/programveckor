## Projekt & Teknisk- beskrivning
ExamShare är en webbaserad plattform framtagen för att hjälpa studenter att studera mer effektivt inför prov och examinationer. Projektets mål är att samla flera viktiga studiefunktioner på ett och samma ställe för att minska stress och skapa struktur i studierna.

Plattformen innehåller bland annat:
- Övningsuppgifter som man kan välja vilket ämne och svårheten mellan lätt - extrem
- Focus Mode aka (fokusläge) med timer för ostört pluggande
- Tillgång till gamla NP
- Möjlighet att ladda upp och spara egna prov

Ett stort fokus i projektet har varit UI/UX. Gränssnittet är designet för att vara enkelt, tydligt och lätt att förstå även för ovana användaren. Navigeringen är konsekvent genom hela webbplatsen och användaren når alla funktioner med få klick. Färgval, knappar och layouten är anpassade för att skapa en lugn och studievänlig upplevelse.

Teknisk beskrivning:
ExamShare använder enkla men effektiva tekniska lösningar för att skapa en smidig användarupplevelse. Ett inloggningsystem gör det möjligt för användare att få tillgång till personliga funktioner som sparade prov och focus mode.

Övningsuppgifter kan filteras efter ämne och svårighetsgrad, vilket hjälper användare att snabbt hitta rätt material. Focus mode innehåller en timer där användare själv ställer in studietid för att kunna arbeta lugn och ro.

Uppladningar av prov gör det möjligt att samla och dela studiematerial på ett strukturerad sätt. De tekniska lösningarna arbetar i bakgrunden och är utformade för att vara enkelt att använda även för personer utan teknisk kunskap.

### Externt producerade komponenter
I projekten så har använde vi standardbibliotek och resurser för webbutveckling, såsom:
- Grundläggande HTML och CSS för struktur och design
- Eventuellt webbläsarbaserade funktioner (t.ex. timers)

Dessa komponenter är nödvändiga för att webbplatsen ska fungera korrekt men utgör endast en grund, den huvudsakliga designen, strukturen och funktionaliteten är egenutvecklad.

### Install
För att kunna köra webbplatsen så behöver vi:
1. Ladda ner projektmappen
2. Öppna mappen (Logga in)
3. Start filen login.html i webbläsre (T.ex. Chrome eller Edge)
4. Webbplatsen körs lokalt och kräver ingen extra instellation
